# 🚀 App Native com Expo + NativeWind

📱 Aplicação mobile desenvolvida com **React Native + Expo**, utilizando **NativeWind (Tailwind CSS)** para estilização.

---

## 🚧 Status do projeto

🟡 Em desenvolvimento
Novas funcionalidades serão adicionadas em breve.

---

## 🛠️ Tecnologias

* ⚛️ React Native
* 🚀 Expo
* 🎨 NativeWind
* 🟦 TypeScript

---

## 📂 Estrutura do projeto

```
src/
 ├── components/
 ├── styles/
 ├── utils/
 └── lib/
```

---

## ▶️ Como rodar o projeto

```bash
git clone https://github.com/Adrianotrabuco/app-nativecn-nativewind.git
cd app-nativecn-nativewind
npm install
npx expo start
```

---

## 📲 Executar no celular

1. Instale o app **Expo Go**
2. Escaneie o QR Code exibido no terminal
